修改WSL默认登录用户

2020-07-17 约 72 字 预计阅读 1 分钟

wsl 安装完成后默认登录是新建用户，切换root用户登录操作如下，然后重新启动wsl即可。

1

ubuntu config --default-user root

ubunt是你的系统，如果不是可以换成你用的系统

Linux创建一个拥有root权限的用户 上一篇
